# 🚀 Freelancer Website

A modern **Freelancer Portfolio Website** built using Astro, React, and Node.js to showcase services, portfolio, and generate client leads.

---

## 🌐 Live Demo
https://freelancer-website-zxt2.b12sites.com/

---

## 📌 About Project

This project is a **full-stack freelancer website** designed to:

- Showcase services and pricing  
- Display portfolio and case studies  
- Collect client inquiries via contact form  
- Build trust using testimonials and blog  

---

## ⚙️ Tech Stack

- **Frontend:** Astro + React  
- **Styling:** Tailwind CSS + shadcn/ui  
- **Backend:** Node.js + Express  
- **Build Tool:** Vite  
- **Icons:** Lucide  

---

## ✨ Features

- 🌙 Dark Mode  
- 📱 Responsive Design  
- 🔍 Blog Search  
- 🎯 Portfolio Filter  
- 📩 Contact Form  
- 🔐 Google reCAPTCHA  
- ⚡ Fast Performance
